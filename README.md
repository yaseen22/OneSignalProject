# OneSignalProject
This app use OneSignal REST APIs for creating, updating and view apps

# Installation
- Clone the solution to your local repo
- Open the solution
- It's preferred to open Web.config and update the Data Source in the connectionStrings node with your local SQL Server name, but if you're using SQL Server Express you can leave it 
- Press F5 to run the project in debug mode or Ctrl + F5 to run the project without debugging
- At the startup a default Admin user is created with these credentails
